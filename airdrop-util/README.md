# airdrop-util

Скрипт для автоматической отправки некоторого количества айрдропов в сети `Solana`.

При использовании срипта важно помнить, что `Solana` оптимизирует запросы: если на один и тот же адрес за короткий промежуток времени отправляется большое количество айрдропов, то все они будут объединены в одну транзакцию. Следовательно, для создания разных транзакций необходимо отправлять айрдропы на разные кошельки (в аргумент `-k` передать JSON-файл с массивом ключей) или выставить задержку (через `-d`) равную примерно одному слоту (~300 мс).

### Опции

- `--version`  
  Показать номер версии
- `-k`, `--key`  
  Публичный ключ кошелька или путь к файлу, где ключи хранятся (если не указано ничего, то будет сгенерировано количество ключей, которое равно желаемому количеству транзакций)
- `-u`, `--url`  
  URL сети
- `-c`, `--cluster`  
  Рабочий кластер (нужно для создания ссылки на транзакцию)
- `-s`, `--supply`  
  Размер айрдропа
- `-l`, `--loop`  
  Количество транзакций
- `-d`, `--delay`  
  Задержка между отправленными транзакциями (мс)
- `-h`, `--help`  
  Показать помощь

### Запуск
1. Клонировать репозиторий:
</br>`git clone https://github.com/relsa228/airdrop-util.git`</br>

2. Установить необходимые зависимости:
</br>`cd airdrop-util && yarn install`</br>

3. Запустить утилиту:
</br>`yarn airdrop -k {WALLET_ADDRESS}`</br>
